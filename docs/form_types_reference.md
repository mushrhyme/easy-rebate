## NET 계산 규칙

| 양식 | 수식 | 비고 |
|------|------|------|
| **01** | `仕切 - (条件 + 条件2)` | CS일 때 아래 참고 |
| **02** | `仕切 - (条件 + 条件2)` | |
| **03** | `仕切 - (条件 + 条件2)` | 条件 없으면 `仕切 - 単価` |
| **04** | `仕切 - (未収条件 + 未収条件2)` | |
| **05** | `仕切 - (個別条件 + 個別条件2)` | |

모든 양식 공통: `NET = 仕切 - (조건1 + 조건2)` (조건2가 없으면 0으로 처리)

### CS 특수 처리 (01번, `cs_divide_by_irisu: true`)

`数量単位 = CS` + `入数 > 0` 이면:
```
NET = 仕切 - ((条件 + 条件2) / 入数)
```

## 특수 처리 규칙

### 소수점 변환 (decimal_conversion)

| 양식 | 대상 필드 | 변환 규칙 | 안전 가드 키 |
|------|----------|----------|-------------|
| **03** | 未収条件 | `value / 100` (예: 370 → 3.70) | 条件％, ケース, バラ |
| **04** | 未収条件 | `value / 100` (예: 1000 → 10.00) | 対象数量又は金額, 未収条件2, 入出荷支店 |

- 값에 이미 소수점(`.`, `．`)이 있으면 변환 생략 (포맷 통일만 수행)
- 가드 키가 하나도 없으면 변환하지 않음 (오변환 방지)

### 행 병합 (row_merging)

| 양식 | 병합 조건 | 동작 |
|------|----------|------|
| **02** | `計算条件（適用人数）` 값이 `数量条件`이 아닌 행 | 해당 행의 조건/금액을 직전 행의 조건2/금액2로 병합 |
