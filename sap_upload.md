주어진 sap_upload.xlsx의 양식에 맞게 분석 결과물을 편집한 엑셀을 생성해야 합니다.

A~BB열까지 존재하지만 
실질적으로 값을 기입해야 하는 컬럼은 다음과 같으며, 양식지 종류에 따라 입력해야 하는 값과 계산식이 달라집니다.
아래 기술한 양식지별 로직이 없을 경우에는 공란으로 두면 됩니다.

====================
[데이터 입력 컬럼]
cover, detail 페이지의 정보를 활용해아 하는 컬럼은 다음과 같습니다.
====================
[B열] 판매처
company_name

[C열] 판매처코드
[D열]
[I열]
[J열]
[K열]
得意先

[L열]
商品名

[P열]
03: ケース数量

[T열]
01:
IF 数量単位="個" → 数量
IF 数量単位="CS" → 入数*数量

02: 取引数量合計（総数:内数）
03: バラ数量
04: 対象数量又は金額

[Z열]
01:
IF 条件区分="個" → 条件
IF 条件区分="CS" → 金額/入数*数量

03: 条件+条件小数部*0.01
04: 未収条件+未収条件小数部*0.01

[AD열]
03: 単価+単価小数部*0.01

[AL열]
01: 金額
02: リベート金額（税別）
03: 請求金額
04: 金額
05: 請求合計額

====================
[엑셀 수식 컬럼]
엑셀 산식이 필요한 컬럼은 다음과 같습니다.
====================

[U열]
=P3*N3 + R3*O3 + T3

[V열]
=U3 / N3

[AF열]
=Z3 + AB3/O3 + AD3/N3

[AG열]
=AA3 + AC3/O3 + AE3/N3

[AH열]
=X3 - AF3 - AG3

[AI열]
=AF3 * U3

[AJ열]
=AG3 * U3

[AK열]
=AL3 - AJ3 - AI3

[AM열]
=AH3 / 0.85

[AP열]
=AH3 - AM3*AN3*0.01 - AM3*AO3*0.01

[AT열]
=X3 - AP3
