PAGE ROLE RULES

The field page_role must be a string and must be one of:

"cover"
"detail"
"summary"
"reply"

page_role must be inferred from the content of the current page.

It is not explicitly written in the document.

page_role decision rules (priority order):

If the page contains repeating item rows such as product / quantity / amount records → "detail"

If the page mainly contains totals, tax, or summary billing information → "summary"

If the page mainly contains company name, address, invoice title, document identifiers,
and no repeating item rows → "cover"

Otherwise → "reply"
